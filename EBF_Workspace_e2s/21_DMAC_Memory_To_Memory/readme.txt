【*】 程序简介 
-工程名称：CGC - 时钟控制
-实验平台: 野火_瑞萨启明6M5开发板
-e2 studio版本：
-FSP固件库版本：4.0.0


【 ！】功能简介：
使用 DMAC 存储器到存储器，传输数据

【 ！】实验操作：
编译并下载程序到开发板并复位，
若红色LED灯亮起，则DTC传输数据失败，若蓝色LED亮起，则DTC数据传输成功

【 ！！】注意事项：
可通过修改 bsp_dmac.h 中的宏 DTC_TRANSFER_MODE 来选择 DMAC 的传输模式

/*********************************************************************************************/

【*】 引脚分配
LED：
LED灯的三个阴极接到6M5三个引脚上，LED低电平亮。

	LED_R<--->P601
	LED_B<--->P602
	LED_G<--->P604
